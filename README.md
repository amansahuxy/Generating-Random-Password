# 🔐 Random Password Generator

A simple Password Generator built using HTML, CSS, and JavaScript. It generates a strong 12-character password containing uppercase letters, lowercase letters, numbers, and special symbols.

## 🚀 Features

* Generate random passwords instantly
* Includes uppercase letters, lowercase letters, numbers, and symbols
* Clean and responsive user interface
* One-click password generation
* Beginner-friendly JavaScript project

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/password-generator.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## 🧠 How It Works

1. The user clicks the **Generate Password** button.
2. JavaScript creates a random password.
3. The password contains:

   * Uppercase letters (A-Z)
   * Lowercase letters (a-z)
   * Numbers (0-9)
   * Special symbols (!@#$%^&*)
4. The generated password is displayed in the input field.
